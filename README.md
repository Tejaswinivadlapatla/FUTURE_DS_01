# Business Sales Performance Analytics Dashboard
## Project Overview
This project focuses on analyzing business sales data to understand overall business performance, identify sales trends, evaluate product profitability, and generate actionable business insights using Microsoft Excel.
## Dataset Source
The dataset used for this project is the Sample Superstore Sales Dataset, commonly used for sales and business analytics practice.
Dataset Link:
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
## The dashboard includes:
- Total Sales Analysis
- Total Profit Analysis
- Total Orders Overview
- Average Sales Calculation
- Monthly Sales Trend Analysis
- Region-wise Sales Comparison
- Segment-wise Sales Distribution
- Highest Revenue Product Identification
- Category-wise Profit Analysis

Interactive slicers were also added for:
- Region
- Category
- Customer Segment

##  Business Insights Generated
The analysis provided several useful business insights, including:
- Certain regions generated higher sales compared to others.
- Technology products produced the highest profit margins.
- Sales showed strong growth during the final months of the year.
- Consumer segment contributed significantly to total revenue.
- Some product categories performed better than others in profitability.

  ## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Conclusion
This project demonstrates how data analytics and visualization can transform raw business data into meaningful insights. The dashboard provides a professional and interactive solution for analyzing sales performance and supports businesses in making smarter, data-driven decisions.
